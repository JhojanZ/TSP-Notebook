# Red Neuronal - TSP con Algoritmo de Hormigas

## Instalación

### Requisitos
- Python 3.12.3+

### Setup rápido

**Opción 1: Usando el script automático**
```bash
./setup.sh
```

**Opción 2: Manual**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requeriment.txt
```

### Activar el entorno (próximas veces)
```bash
source venv/bin/activate
```

### Ejecutar el proyecto
```bash
jupyter notebook parcial.IPYNB
```
